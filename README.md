I am an exteremely content owner of the [TRMNL device](https://trmnl.com/?ref=Itsfoss10).

While they have plenty of official third-party plugins in their marketplace, their developer edition allows adding custom plugins.

This is perfect for people who would like to use TRMNL to display stuff that matters to them. 

I have created a few plugins for my own perosnal use. Probably I'll create some more in the future. I have used AI assistance in coding.

If you like these plugins, you can download the code, modify it if required and use it on your TRMNL device.

## How to use these plugins?

Each plugin has a zip file that can be imported as a private plugin. You must have purchased the developer license which costs a one-time fee of $20 for that. 

Go to your TRMNL dashboard and go to private plugin. You should see the option to Import or Add New.
